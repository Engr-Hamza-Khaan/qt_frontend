# 🌟 React Admin Dashboard

A modern, responsive **React Admin Panel** featuring sidebar navigation, dark/light mode, charts, analytics, and activity tracking — all crafted with **Tailwind CSS** for a clean, elegant UI.

---

## ✨ Key Features

### 🧭 Sidebar Navigation
- Fully responsive sidebar with expand/collapse animation  
- Submenu toggle functionality with smooth transitions  
- Active link highlighting and elegant hover effects  

### 🌗 Dark & Light Mode
- Built-in theme toggle for dark and light modes  
- Adaptive colors and shadows for a polished UI look  

### 🔍 Interactive Header
- Search bar for quick access  
- Notification, settings, and profile menu  
- Smooth layout & clean design  

### 📊 Dashboard Overview
- Analytics section displaying key performance indicators  
- Charts and Graphs including bar chart and pie chart for insights  
- Orders Table showcasing latest orders with dynamic status styles  
- Top Products Section displaying best performing products with trends  
- Activity Log showing recent admin or user actions  

---

## 🧰 Tech Stack

| Technology | Description |
|-------------|-------------|
| ⚛️ **React.js** | Component-based UI framework |
| 🎨 **Tailwind CSS** | Modern styling with utility-first classes |
| 💡 **Lucide-react** | Clean and customizable icons |
| 📈 **Recharts** | For interactive and beautiful data visualization |
| 🌙 **Dark/Light Mode Toggle** | Built using Tailwind’s dark variant |
